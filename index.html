Virtual Pet

Virtual Pet

  • A Java-based pet shelter that allows users to create, interact with, and delete virtual creatures from a digital pet shelter.

  • OOP principles were used to create templates for both robotic and organic pets of different species.

  • This project has gone through multiple iterations, starting off with a single object(pet), and then added functionality to accommodate multiple pets and species.

To-Do App

To-Do App

  • A simple task tracker app. Add an outstanding task to the list, mark them as complete as you go, and remove them.

  • Using vanilla JavaScript to allow the user to interact with the list, functionality enabled by Event Listeners.

  • Basic HTML/CSS was used to structure the web app in a simple, yet clean design.

Adagio Album Collection

Adagio Album Collection

  • An album collection that utilizes Spring MVC, creating a fullstack application using Javascript to display albums and songs, as well as individual ratings. Backend mapping allows the user to add, edit or delete songs or albums from the collection.

  • JSON data used to populate a Java backend, enabling management of all stored album and song objects.

  • The frontend was made with JavaScript with DOM design principles to create a light and dynamic single page application.

Donut Clicker

Donut Clicker

  • A cookie clicker-inspired game utilizing Javascript to allow the player to click a button that dynamically increases the score.

  • The game also features the ability to increase the base multiplier of donuts gained per click, as well as Auto-Clickers, which will automatically increase the score on a set interval tracked by the game. These funcionalities are enabled by Event Listeners.
